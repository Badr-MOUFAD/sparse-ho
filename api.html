Models¶

sparse_ho.models:

Lasso([max_iter, estimator, log_alpha_max])

Linear Model trained with L1 prior as regularizer (aka the Lasso) The optimization objective for Lasso is: (1 / (2 * n_samples)) * ||y - Xw||^2_2 + alpha * ||w||_1

WeightedLasso([max_iter, estimator, …])

Linear Model trained with weighted L1 regularizer (aka weighted Lasso)

SVM([estimator, max_iter])

The support vector Machine classifier without bias The optimization problem is solved in the dual: 1/2 r^T(y * X)(y * X)^T r - sum_i^n r_i s.t 0 <= r_i <= C
