Models¶

sparse_ho.models:

Lasso(X, y, log_alpha[, log_alpha_max, …])

Linear Model trained with L1 prior as regularizer (aka the Lasso) The optimization objective for Lasso is: (1 / (2 * n_samples)) * ||y - Xw||^2_2 + alpha * ||w||_1

wLasso(X, y, log_alpha[, log_alpha_max, …])

Linear Model trained with L1 prior as regularizer (aka the weight Lasso) The optimization objective for weighted Lasso is: (1 / (2 * n_samples)) * ||y - Xw||^2_2 + sum_i^n_features alpha_i |wi|

SVM(X, y, logC[, max_iter, tol])

Support vector machines.

Algorithms¶

ImplicitForward(criterion[, tol_jac, …])

Algorithm that will compute the (hyper)gradient, ie the gradient with respect to the hyperparameter using the implicit forward algorithm.

Forward(criterion[, use_sk, verbose])

Algorithm that will compute the (hyper)gradient, ie the gradient with respect to the hyperparameter using the forward algorithm.

Backward(criterion)

Algorithm that will compute the (hyper)gradient, ie the gradient with respect to the hyperparameter using the bacward differentiation.
